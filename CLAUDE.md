## Project Overview

BAT VR sales training demo. Practice pitching to AI customers via real-time voice.

## Tech Stack

- **Next.js 16** with App Router
- **TypeScript**
- **Tailwind CSS v4**
- **OpenAI Realtime API** - WebSocket voice conversations
- **Convex** - Backend

## Commands

```bash
npm run dev      # Dev server (localhost:3000)
npm run build    # Production build
npm run lint     # ESLint
vercel --prod    # Deploy
```

## Key Features

- 5 customer personas
- Supervisor model for evaluation
- PDF report cards
